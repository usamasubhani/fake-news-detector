How it Works:

This tool can be used to detect clickbait using stance detection.
It extracts Term Frequencies from user's input, calculates Inverse Documunt Frequencies of the headline and body.
Term Frequencies and Cosine Similarity betweeen TF-IDF is concatenated in a vector which
is fed to a trained Neural Network which outputs the result.
